# Rock, Paper, Scissors

Browser console run game of Rock, Paper, Scissors.

Plays a game of 5 rounds, each round you will enter your choice of rock, paper, or scissors.

![Move Input](/screenshots/move_input.png)

At the end of 5 rounds, whoever finished with the most points wins.

View results in the browser console.
